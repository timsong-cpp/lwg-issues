2247. Type traits and std::nullptr_t

Section: 20.15.5.2 [meta.unary.cat] Status: C++14 Submitter: Joe Gottman Opened: 2013-03-15 Last modified: 2016-01-28 10:19:27 UTC

Priority: Not Prioritized

View all issues with C++14 status.

Discussion:

According to 20.15.5.2 [meta.unary.cat], for every type T, exactly one of the primary type traits is true. So which is true for the type std::nullptr_t? By 5.13.7 [lex.nullptr] std::nullptr_t is not a pointer type or a pointer-to-member type, so is_pointer, is_member_object_pointer and is_member_function_pointer can't be true for std::nullptr_t, and none of the other primary type traits seem to apply.

[2013-04-20, Bristol]
