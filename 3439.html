Discussion:

The library specification conveys pre-concept requirements on template parameters by naming, e.g. "InputIterator".

There is no general specification for template parameters named "Distance", as used by std::advance (see 23.4.2 [iterator.operations]). When addressing this omission, the local restriction in 25.7.12 [alg.random.sample] could possibly be removed.

See the related issue 3213 for the Size template parameter.
