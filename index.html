Like LWG's official repository, with a few changes:
  • Pages for individual issues, like https://timsong-cpp.github.io/lwg-issues/1234. This has been merged upstream.
  • Inter-issue links go to the individual issue pages rather than the master list.
  • Section references to a stable name present in the current working draft are automatically linked to my HTML version.Upstream now linkifies those tags too.
  • Section reference links go to my HTML version, rather than wg21.link (which in turn goes to eel.is).
  • Issue timestamp reflects the timestamp of the last git commit that touched the issue, making it slightly less useless.
  • List timestamp is based on the most recent issue timestamp, rather than the time at which the generation script is run.
