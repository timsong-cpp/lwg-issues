2868. Missing specification of bad_any_cast::what()

Section: 19.8.2 [any.bad_any_cast] Status: C++17 Submitter: Great Britain Opened: 2017-02-03 Last modified: 2018-08-20 12:41:52 UTC

Priority: Not Prioritized

View all issues with C++17 status.

Discussion:

Addresses GB 54

There is no specification for bad_any_cast.what.

Proposed change:

Add a paragraphs: 

const char* what() const noexcept override;

Returns: An implementation-defined NTBS.

Remarks: The message may be a null-terminated multibyte string (17.5.2.1.4.2), suitable for conversion and display as a wstring (21.3, 22.4.1.4).

[Kona 2017-03-01]

Accepted as Immediate to resolve NB comment.

Proposed resolution:

This wording is relative to N4618.

  1. Insert the following series of paragraphs to 19.8.2 [any.bad_any_cast] as indicated:

    const char* what() const noexcept override;

    -?- Returns: An implementation-defined NTBS.

    -?- Remarks: The message may be a null-terminated multibyte string (15.4.2.1.5.2 [multibyte.strings]), suitable for conversion and display as a wstring (20.3 [string.classes], 26.4.1.4 [locale.codecvt]).
