124. ctype_byname<charT>::do_scan_is & do_scan_not return type should be const charT*

Priority: Not Prioritized

View all other issues in [locale.ctype.byname].

Discussion:

In Section 28.3.4.2.3 [locale.ctype.byname] ctype_byname<charT>::do_scan_is() and do_scan_not() are declared to return a const char* not a const charT*.

Proposed resolution:

Change Section 28.3.4.2.3 [locale.ctype.byname] do_scan_is() and do_scan_not() to return a const charT*.
