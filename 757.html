757. Typo in the synopsis of vector

Section: 21.3.11 [vector] Status: NAD Editorial Submitter: Paolo Carlini Opened: 2007-11-04 Last modified: 2016-01-28 10:19:27 UTC

Priority: Not Prioritized

View all other issues in [vector].

View all issues with NAD Editorial status.

Discussion:

In the synopsis 21.3.11 [vector], there is the signature: 

void insert(const_iterator position, size_type n, T&& x);

instead of: 

iterator insert(const_iterator position, T&& x);

21.3.11.5 [vector.modifiers] is fine.

Proposed resolution:

Change the synopsis in 21.3.11 [vector]: 

iterator insert(const_iterator position, const T& x); 
iterator insert(const_iterator position, T&& x);
void     insert(const_iterator position, size_type n, const T& x); 
void     insert(const_iterator position, size_type n, T&& x);
