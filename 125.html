125. valarray<T>::operator!() return type is inconsistent

Section: 26.7.2 [template.valarray] Status: TC1 Submitter: Judy Ward Opened: 1998-12-15 Last modified: 2016-01-28 10:01:27 UTC

Priority: Not Prioritized

View all other issues in [template.valarray].

View all issues with TC1 status.

Discussion:

In Section 26.7.2 [template.valarray] valarray<T>::operator!() is declared to return a valarray<T>, but in Section 26.7.2.6 [valarray.unary] it is declared to return a valarray<bool>. The latter appears to be correct.

Proposed resolution:

Change in Section 26.7.2 [template.valarray] the declaration of operator!() so that the return type is valarray<bool>.
